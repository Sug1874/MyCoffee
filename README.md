# My Coffee

## Overview
You can use this application to record and refer to information and impressions of the coffee you drink.

## How to run
1. Clone this repository
```bash
git clone https://github.com/Sug1874/MyCoffee.git
```

2. Change current directory
```bash
cd MyCoffee
```

3. Set up 
```bash
bundle install
rails db:migrate
```

4. Start application
```bash
rails server
```
